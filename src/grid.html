Wrapping grid

Wrapping grids allow cells to flow onto another line. Since there is no CSS selector for a wrapped cell, cells in a wrapped grid have their margins removed to avoid odd-looking indents on wrapped lines.

Column Basis

Grid cells can be given an initial "main size" before free space is distributed according to other flex factors. If flex items' basis' exceeds free space (e.g. you define 14 columns in the 12 column space), then flex items will still continue to shrink.
